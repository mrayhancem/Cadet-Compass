Credits and Source Attribution

This project is a Track III (Educator) submission for the Presidential AI Challenge. It combines educator-authored guidance with official-source verification links and an AI Coach (optional) that provides safe, general advising.

Important: Requirements can change. Cadet Compass always instructs users to verify details on official sources.

AI tools and platforms

  • OpenAI API (text model via a Netlify Function) for the optional “AI Coach” responses.
  • Netlify for hosting and serverless functions (API endpoint for chat).
  • GitHub for version control and public source repository.
Note: The site remains usable even without the AI function enabled, via a local knowledge-base fallback (data/knowledge.json).

Primary sources used (official)

Cadet Compass links to and/or is informed by official program pages such as:

  • U.S. Service Academy admissions pages (USMA, USNA, USAFA, USMMA, USCGA).
  • ROTC official program pages (Army ROTC, Navy ROTC, Air Force ROTC).
  • Military recruiting and enlistment information pages (branch recruiting sites).
  • ASVAB information resources (official ASVAB program and DoD-related references).
The “Student Toolkit” page consolidates official links so students can verify eligibility, steps, and timelines.

Graphics and design

  • Logo: Compass mark (SVG) included in project assets (assets/compass.svg).
  • Site UI: Custom HTML/CSS (assets/styles.css) and JavaScript.
