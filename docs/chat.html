AI Coach (Cadet Coach)

Guidance tool for Grades 8–12. It prioritizes safe advising: general planning, pathway comparisons, and official-source verification. It avoids “hard numeric promises” and tells you what must be confirmed.

Privacy: Do not enter sensitive personal data (full name, address, SSN, medical details). Use general details only (grade level, pathway interest, general goals).

If the Netlify Function is configured, the coach uses AI + the site’s local guidance. If not configured, it runs purely offline with local guidance.

How to use this tool

  • State your grade (8–12) and pathway interest (Academy / ROTC / Guard-Reserve / Enlistment).
  • Ask for a semester plan (not a vague “what should I do”).
  • Ask for a Plan B and risk flags.
  • Verify requirements using official links in Student Toolkit.
Scope guardrails:
This coach provides educational guidance only. It does not provide operational or unsafe instructions. It does not replace parents/guardians, counselors, or official program staff.
Recommended prompt format
Grade: __ Texas student: Yes Pathway interest: __ (Academy / ROTC / Guard-Reserve / Enlistment) Goal: __ (officer, scholarship, job training, etc.) Constraints: __ (time, transportation, family responsibilities) Request: "Give me a plan for the next 60 days + risks + what to verify."
